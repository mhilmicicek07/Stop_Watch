## Türkçe (TR)

### Hakkında
Basit, kullanımı kolay ve tarayıcı üzerinde çalışan bir Kronometre (Stop Watch) uygulamasıdır. HTML, CSS ve JavaScript ile geliştirilmiştir.

### Özellikler
- ▶️ Başlat / Durdur
- ⏸️ Duraklatma — durdurduktan sonra kaldığı yerden devam eder
- 🔁 Sıfırlama
- ⏱️ Milisaniye, saniye ve dakika gösterimi
- 💻 Hafif ve modern kullanıcı arayüzü

### Kullanılan Teknolojiler
- HTML5
- CSS3
- JavaScript (ES6+)

### Proje Dosya Yapısı
Stop_Watch/
├── index.html  
├── style.css  
├── script.js  
└── README.md

### Kurulum ve Çalıştırma
1. Depoyu klonlayın:
```bash
git clone https://github.com/mhilmicicek07/Stop_Watch.git
cd Stop_Watch
```
2. index.html dosyasını herhangi bir modern tarayıcıda açın (Chrome, Firefox, Edge, Safari).

Hepsi bu kadar — kronometre hazır!

### Teknik Notlar
- Zaman güncellemesi `setInterval()` kullanılarak yapılıyor.
- Kronometreyi durdurmak için `clearInterval()` kullanılıyor.
- Reset fonksiyonu tüm sayaç değerlerini sıfırlar ve UI'ı günceller.

## English (EN)

### About
A simple, easy-to-use browser-based Stop Watch application built with HTML, CSS and JavaScript.

### Features
- ▶️ Start / Stop
- ⏸️ Pause/Resume — stops and continues from the same time
- 🔁 Reset
- ⏱️ Shows minutes, seconds and milliseconds
- 💻 Lightweight, modern UI

### Technologies
- HTML5
- CSS3
- JavaScript (ES6+)

### Project Structure
Stop_Watch/  
├── index.html  
��── style.css

### Installation & Run
1. Clone the repository:
```bash
git clone https://github.com/mhilmicicek07/Stop_Watch.git
cd Stop_Watch
```
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

That's it — the stopwatch is ready to use!

### Technical Notes
- Time updates are handled using `setInterval()`.
- The stopwatch is stopped using `clearInterval()`.
- The reset function clears all time values and updates the UI.

## Deutsch (DE)

### Über
Eine einfache, benutzerfreundliche Stoppuhr-Anwendung, die im Browser läuft. Entwickelt mit HTML, CSS und JavaScript.

### Funktionen
- ▶️ Start / Stopp
- ⏸️ Pause / Fortsetzen — hält an und setzt an der gleichen Stelle fort
- 🔁 Zurücksetzen
- ⏱️ Anzeige von Minuten, Sekunden und Millisekunden
- 💻 Leichtgewichtiges, modernes UI

### Technologien
- HTML5
- CSS3
- JavaScript (ES6+)
